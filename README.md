## What I Build

- **AI Agents & Multi-Agent Systems**<br>
  Custom LLM integrations with supervisor orchestration, triple-layer memory, and 60+ specialized agents built around real operational problems

<br>

- **Security Assessment Tools**<br>
  Open-source Windows security tooling with MITRE ATT&CK tagging (see: *Am I Hacked?*)

<br>

- **Steam Games & Steamworks Integration**<br>
  Published *System.Execute* (AppID 4353080) on Steam, an Electron-based incremental idle game with full Steamworks SDK integration including achievements, Steam Cloud saves, global leaderboards via Firebase, Steam Overlay profile linking, and SteamPipe deployment pipelines

<br>

- **Automation Pipelines**<br>
  Connecting CRM, data, and operational tools across platforms (Monday.com, Microsoft Graph, SharePoint, Supabase)

<br>

- **MCP Ecosystem & Tooling**<br>
  Docker-based MCP Lab environments with Kali Linux, Obsidian, and custom server integrations

<br>

- **Nonprofit & Operational Tech**<br>
  Production systems serving 350,000+ people monthly: Teams rollouts, SharePoint automation, and large-scale logistics tracking

<br>

- **Homelab Infrastructure & Automation**<br>
  Full Raspberry Pi 5 homelab managed entirely from Windows, using Ansible playbooks via Docker, PowerShell monitoring/maintenance scripts, an Electron desktop app, and a self-hosted service stack (Grafana, Prometheus, AdGuard, Vaultwarden, Portainer, Tailscale VPN, and 15+ others)

<br>

- **Hospitality & SMB Tech Consulting**<br>
  AI integration advisory, POS systems, and operational tooling through TM Hospitality Strategies, AKA TMHS or TMHSDigital (DBA)
